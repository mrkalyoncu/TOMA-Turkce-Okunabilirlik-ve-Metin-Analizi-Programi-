# TOMA-Turkce-Okunabilirlik-ve-Metin-Analizi-Programi


TOMA programını çalıştırmak için "TurkceOkunabilirlik" klasörünü indiriniz. 

Programı çalıştırabilmek için bilgisayarınızda "Eclipse Temurin JDK with Hotspot 25.03+9" Java programının kurulu olması gerekmektedir. Programın .exe dosyası klasör içinde yer almaktadır. 

İlgili programı indirdikten sonra "TurkceOkunabilirlik" uygulamasını çalıştırarak analizlerinizi gerçekleştirebilirsiniz. 

Sorularınız ve destek talepleriniz için mrkalyoncu@sinop.edu.tr adresinden iletişime geçebilirsiniz.


PROGRAMI AKADEMİK BİR ÇALIŞMADA KULLANIYORSANIZ ÇALIŞMAYI REFERANS GÖSTERMENİZ GEREKMEKTEDİR.
